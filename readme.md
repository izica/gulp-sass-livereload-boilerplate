## Features
* SASS/SCSS minifing
* JS minifing
* Images optimizing
* LiveReload dev server

## Install
```bash
    npm install
```

## Usage
```
npm start
```

Open http://localhost:8080

## Commands
```
    npm start //start livereload server and watch
    npm run build //build sass+js
    npm run images //images
    npm run css //build sass
    npm run js //build js
```
